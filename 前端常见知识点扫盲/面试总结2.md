# 面试题目总结第二部分 -- Vue


### Vue

#### 实现数据双向绑定

1、Object.definedProperty

```
var person = function() {
    var _name = ' ';
    var obj = {};
    Object.defineProperty(obj, 'name', {
        configurable: true,
        enumerable: true,
        get: function() {
            return _name;
        },
        set: function(n) {
            _name = n;
        }
    })
    return obj;
}();
person.name = "Zhangsan";
person.name // 输出 --> Zhangsan
```
